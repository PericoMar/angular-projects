Tecnologías

Angular ha sido el principal framework utilizado en el desarrollo de la parte front de mi aplicación. Este potente framework de desarrollo frontend ha proporcionado una estructura sólida y modular para la construcción de una experiencia de usuario dinámica y receptiva.

Uno de los aspectos más destacados del uso de Angular ha sido su capacidad para gestionar las solicitudes y respuestas de la API de manera eficiente. Los servicios de Angular han sido fundamentales para establecer la comunicación con la API REST Countries, permitiendo que mi aplicación obtenga y muestre información detallada de países de todo el mundo en tiempo real.

Además, Angular's Router Module ha sido esencial para la gestión de las rutas de mi aplicación. Con él, he podido definir y organizar las diferentes vistas y componentes de mi aplicación, facilitando la navegación del usuario a través de las diversas secciones y funcionalidades.

En resumen, Angular ha sido una pieza fundamental en el desarrollo de mi aplicación, proporcionando las herramientas necesarias para crear una experiencia de usuario fluida y atractiva, así como para establecer una sólida conexión con la API que impulsa mi aplicación de búsqueda de países.
